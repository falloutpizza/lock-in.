# lock in.

lock in is a firefox add-on that lets users start timed study sessions with breaks that can be selected to run after each session.
its features include:
- three timed modes: study, short break, and long break
- an interface to change current timer mode, reset the timer, or change default timer settings
- an onscreen timer that shows on the currently opened tab even when the interface is closed

download it here: https://addons.mozilla.org/en-US/firefox/addon/lock-in-a-productivity-tool/
